# 🤖 AI Interview Preparation Assistant

## 📌 Project Overview

AI Interview Preparation Assistant is an AI-powered web application developed using Streamlit and Google's Gemini API. It helps users practice interview questions and receive instant feedback, including scores, strengths, weaknesses, and suggestions for improvement.

## ✨ Features

- User-friendly web interface
- Multiple interview categories:
  - HR
  - Python
  - AI
  - SQL
  - Java
  - DSA
- AI-powered answer evaluation using Gemini
- Displays:
  - Score
  - Strengths
  - Weaknesses
  - Suggestions
- Calculates average interview score
- Shows overall performance
- Downloadable interview report
- Optional resume upload

## 🛠 Technologies Used

- Python
- Streamlit
- Google Gemini API
- Regular Expressions (re)

## 🚀 Installation

1. Install Python.
2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run the application:

```bash
streamlit run app.py
```

## 📊 Workflow

1. Enter your name.
2. Upload resume (optional).
3. Select interview category.
4. Answer interview questions.
5. Click **Evaluate Interview**.
6. Receive AI-generated feedback.
7. View final score and performance.
8. Download the interview report.
